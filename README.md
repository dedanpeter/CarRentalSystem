Car Rental System

This project implements a simple Car Rental System in Java, utilizing Object-Oriented Programming (OOP) principles.

Features

User Authentication**: Login with predefined usernames and passwords.
Car Management**: View available cars for rent.

 Getting Started

 Prerequisites

Make sure you have the following installed on your system:
Java Development Kit (JDK)

How to Run

1. Clone this repository or download the project files.
2. Open a terminal and navigate to the project directory.
3. Compile the Java files by running:
   bash
   javac *.java
